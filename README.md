It contains all the supporting project files necessary to work through the video course from start to finish.
## About the Video Course
Learn how to style websites. A step by step process is used to show everything from setting up to essential CSS features. Gain a good understanding of the following concepts: CSS programming, Web styling, Basic/essential CSS properties, CSS values, Dynamic website creation. CSS is one of the core universal programming languages of the web that is essential for any stylish web development journey. The web is growing on a daily basis and knowing CSS is more crucial than ever. You will be provided with the full source code to aid in development during and after this course. The source code is free to use in as many projects as you wish.
